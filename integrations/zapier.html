How to Set Up

Get screenshot automation running in minutes

1

Get Your API Key

Sign up for a GetScreenshot Hamilton or Jackson plan. After subscribing, you'll receive an API key via email that includes Zapier access.

View pricing plans
2

Create a Zap with GetScreenshot

In Zapier, create a new Zap and search for "GetScreenshot" when adding an action. Connect your account using your API key.

3

Configure Screenshot Options

Enter the URL you want to capture (this can come from a trigger or previous step). Configure options like viewport size, full page capture, PDF format, and more.

Available Options in Zapier:

  • URL to screenshot
  • Viewport width & height
  • Full page capture
  • PDF output
  • Hide chat widgets
  • Send to email
4

Use the Screenshot in Subsequent Steps

GetScreenshot returns a screenshot URL that you can use in the next steps of your Zap. Save to Google Drive, post to Slack, add to a database, send via email, and more.
