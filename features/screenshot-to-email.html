Screenshot to Email

Send website screenshots directly to any email address. Perfect for automated reports, monitoring alerts, and sharing captures without additional infrastructure.

Get Started - From $5/mo Try Live Demo

How Screenshot to Email Works

Simply add the email parameter to your API request with a valid email address. GetScreenshot will capture the screenshot and deliver it directly to that inbox as an attachment.

The email includes the screenshot as an attachment along with metadata like the URL, capture timestamp, and image dimensions. No need to set up your own email server or file storage.

API Parameter

email={email address}

Pass any valid email address. The screenshot will be sent as an attachment to that address.

Example Request

https://api.rasterwise.com/v1/get-screenshot
  ?apikey=YOUR_API_KEY
  &url=https://example.com/dashboard
  &email=reports@yourcompany.com
  &fullpage=true

Email Contents

Subject: Screenshot of example.com/dashboard

Attached: screenshot_2026-01-18_143022.png

URL: https://example.com/dashboard

Captured: January 18, 2026 at 2:30 PM

Dimensions: 1280 x 2400px
