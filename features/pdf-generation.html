How URL to PDF Works

GetScreenshot's PDF generation feature renders any webpage in a real browser and converts it to a properly formatted PDF document. The result preserves layouts, fonts, images, and styles.

Unlike simple HTML-to-PDF converters, our API handles JavaScript-rendered content, lazy-loaded images, and complex CSS layouts. You get the page exactly as it appears in a browser.

API Parameter

pdf=true

Set to true to receive a PDF instead of an image. The response will include a URL to the generated PDF document.

Example Request

https://api.rasterwise.com/v1/get-screenshot
  ?apikey=YOUR_API_KEY
  &url=https://example.com/invoice
  &pdf=true

Response

{
  "screenshot": "https://cdn.rasterwise.com/...pdf",
  "format": "pdf"
}
