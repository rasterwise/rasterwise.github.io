Hide Chat Widgets

Automatically remove chat widgets from Intercom, Drift, Tawk, Zendesk, Facebook Messenger, and other popular providers. Get clean, professional screenshots with a single parameter.

Get Started - From $5/mo Try Live Demo

How It Works

Chat widgets are essential for customer support, but they can clutter screenshots. The hidechatwidgets parameter automatically detects and hides the most popular chat solutions.

No need to write custom CSS or JavaScript to hide these elements. GetScreenshot maintains an updated list of chat widget selectors and handles the removal automatically.

API Parameter

hidechatwidgets=true

Set to true to automatically hide all recognized chat widgets. Default is false.
