Keyword Highlighting

Automatically highlight keywords and phrases in your screenshots. Perfect for SEO audits, content analysis, brand monitoring, and quality assurance.

Get Started - From $5/mo Try Live Demo

How Keyword Highlighting Works

Pass keywords to the highlight parameter and GetScreenshot will automatically find and highlight all occurrences on the page with a yellow marker style.

The highlighting is applied before the screenshot is taken, making it easy to visually identify where specific terms appear on any webpage.

API Parameter

highlight={keyword or phrase}

Pass the text you want highlighted. Supports single words and multi-word phrases. URL-encode spaces and special characters.

Example: Highlight Brand Name

https://api.rasterwise.com/v1/get-screenshot
  ?apikey=YOUR_API_KEY
  &url=https://techcrunch.com
  &highlight=artificial%20intelligence
  &fullpage=true
