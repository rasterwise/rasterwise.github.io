Awesome Features = Awesome Screenshots

We provides a lot of cool features. Some of them are pretty standard, some are pretty dope.

Custom CSS & JS

GetScreenshot can insert CSS rules and JS scripts to modify the final look and rendering of the page you are screenshooting.

Hide Cookie Disclaimers

GetScreenshot can hide those pesky fixed cookie dialogues that ruin the rendering of your pages.

Hide Message Clients

GetScreenshot can also hide customer support message clients from all the major providers (Intercom, Drift, Tawk, FB, etc.)

Highlight Keywords

GetScreenshot can highlight keywords and phrases. Just pass the keyword and the screenshot will include the visual highlights.

Generate PDF

Besides the standard JPG and PNG formats, GetScreenshot can also create PDFs of the rendered websites.

Screenshot Message

GetScreenshot can send an email message with the taken screenshot as an attachment if you pass the email message parameter.

Webhook

GetScreenshot can simplify your implementation by sending a POST request to your own custom endpoint.
